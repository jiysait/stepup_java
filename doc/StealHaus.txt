3. 객체지향 특정 3가지의 정의를 말해보시오
    Inheritance
        Inheritance is a technique to make one object acquiring the properties of another object.
    Encapsulation
        Encapsulation is defined as the wrapping up of data under a single unit. It is the mechanism that binds together code and the data it manipulates. In Encapsulation, the variables or data of a class are hidden from any other class and can be accessed only through any member function of their class in which they are declared.
    Abstraction
        It is 'Exposing' the necessary/essential details and common characteristics, 'Postponing' other details
    Polymorphism
        Polymorphism is a feature that allows a single interface to be used for the general class of actions.
